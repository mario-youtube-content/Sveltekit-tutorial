# Sveltekit Favorite App

This is a tutorial for svelte kit

Data types:
- Food:
	id: string
	title: string
	tags: string[]
	description: string
	img_url: string
- Quote: string
- Videos:
	url: string
	reason: string

## PART 2 : Favorite Quote

## PART 3 : Favorite Videos
